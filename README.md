🌟 About Project

SIAM BOT V1 is a modern and intelligent Messenger Chat Bot built using Node.js.

Designed for speed, stability and automation, SIAM BOT V1 provides a smooth chatting experience with advanced smart reply systems and powerful management tools.

---

✨ Features

🤖 Smart AI System

- AI Powered Auto Reply
- SimSimi Integration
- Intelligent Conversation Handling

🌍 Multi Language

- Bangla Support
- English Support
- Easy Language Switching

⚡ Performance

- High Speed Response
- Optimized Memory Usage
- Stable Runtime Environment

🛡️ Security
